Program Structure of C++

Basics/Overview

Every C++ Program will follow this basic layout. Includes will be at the top so the compiler can see what librarys are included. Additional incantations just below (using namespace std;). Function prototypes will be below those (Or functions depending on your preference but function prototypes are the norm). Global variables will be below that; variables should be declared with a value (Like 0). Below that there will be main(). Below main will be the function defintions, inside each function that is not a void they must end in a return with the return of the variable type of the function. Every line must be ended in an ';' besides when a brace is needed.
Ex:

#Include <iostream>

using namespace std;

void welcome();

int number = 10;

int main(){
   welcome();

   cout << number;

   return 0;
}

void welcome{
   cout << "Hello World" << endl;

}

Output:

Hello World
10

#Include

You will use #include whenever you want to include a library or other file in your program. For your own file the file will be inside double quotes ("") Librarys the standard Library will be inside arrows pointing opposite directions from each other (<>)
Ex:

#include "myfile.h"

And

#include <iostream>

Using: 'using namespace std;'

Without 'using namespace std;' whenever you use a function, variable, or anything from another class you must first use std::
Ex:

std::cout << "Hello World";

However if you use 'using namespace std;' the std:: is not needed anywhere so

std::cout << "Hello World";

Becomes

cout << "Hello World";
