Basics and Common Mistakes

Indexing

C++ Indexes starting at 0. If I have an character array name[10] Which has John inside it the J in John will be at 0. So if I wanted to output o in John I would do:

cout << name[1];

Output:

o

Comparing Floating-Points

Floating points are any decimal number. You don't want to compare floating points directly because the way the numbers are held are inprecise
Do NOT do:

x == 0.00011112;

Instead do:

x < 0.00011112;

OR

x > 0.00011112;
